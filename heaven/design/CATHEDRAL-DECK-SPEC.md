## DESIGN UPGRADES (Claude's NOIZY additions)

1. **Wisdom Gold `#C49000`** added to palette — elder wisdom feels aged, earned, sacred. Not the same amber as a CTA.
2. **DreamChamber slide** — not a static UI mock. Full-bleed immersive: dark room, sound waves from a human figure, light reacting to presence. The slide *is* the Chamber.
3. **Plowman Standard slide** — "75/25. In Perpetuity. Forever." Cormorant Bold 72px amber on pure black. Nothing else. Heartbeat.
4. **Epochs slide** — visual gravity inversion. I-IV pull downward. V rises. The Elevation Doctrine made visual.
5. **Team/Rob slide** — "40 years. 888 titles. Broken hands. Never stopped." One line.

## PRODUCTION NOTES

### Export Settings
- **Presentation:** 16:9, 1920x1080 or 3840x2160
- **Video:** 30fps, H.264, 1080p minimum
- **PDF:** High-res, bleed marks if printing

### Fonts to License/Install
- Cormorant (Google Fonts — free, commercial use OK)
- Inter (Google Fonts — free, variable weight)
- JetBrains Mono (for any code displays)

### Image Commissions Needed
- Slide 3: Cathedral interior (commission or AI-generate with provenance)
- Slide 6: DreamChamber immersive room (must be original — this IS the product)
- Slide 7: Sound wave tree visualization
- Slide 11: Rob's portrait (professional, studio, warm amber lighting)

### CSS Variables
All colors, fonts, spacing, and animation curves are defined in `design-system.css`. Use these tokens in every NOIZY frontend:
- noizy.ai landing page
- The Aquarium dashboard
- NOIZYVOX artist portal
- DreamChamber browser app
- VS Code Insiders theme (NOIZYBEAST)
