# DREAMCHAMBER FLOW

> Creation journey, approval journey, guidance moments.

## The Creation Journey

1. **Enter** — Artist opens DreamChamber. Gabriel is present (top-right).
2. **Create** — Record, perform, improvise. Gabriel monitors tone, suggests modes.
3. **Capture** — Voice DNA, emotional modes, character archetypes stored.
4. **Review** — Gabriel surfaces insights: "You are strongest in restrained tension."
5. **Publish** — Artifact goes through generate-evidence pipeline.

## The Approval Journey

1. **Client submits script** → Gabriel context-analyzes
2. **Artist rules checked** → boundaries, never clauses, personal limits
3. **Consent validated** → token check, scope check, licensee standing
4. **Humanity Weight scored** → craft + intention + consent_depth + soul
5. **Decision rendered** → GENERATE / ASK_ARTIST / BLOCK / INHERIT
6. **If GENERATE** → synthesize → provenance sign → ledger write → payout route
7. **If ASK_ARTIST** → notification to artist, awaiting approval
8. **If BLOCK** → logged with reason, no synthesis occurs
9. **If INHERIT** → routed to estate beneficiary chain

## Gabriel Guidance Moments

**Tone Drift:**
> "Your tone is drifting toward aggression. Adjust — or save as a new mode?"

**Boundary Conflict:**
> "This script conflicts with your boundary: Political persuasion. Recommend: Reject."

**Below Threshold:**
> "This use is lawful but below Elevation threshold. Approval required."

**Artist Growth:**
> "Your emotional range has expanded 23% since joining. Consider a new character."

**Estate Authority:**
> "Estate authority required. Routing to your beneficiary/trust controller."
