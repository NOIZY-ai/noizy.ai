# NOIZY Cathedral Web — Next.js Landing Page

> The public face of the Cathedral. Black + Amber + Cyan.
> Deep Space Black `#050505`, Amber `#FF9F1C`, Cyan `#00E5FF`

## Sections

1. **Hero** — full screen, noise drift + amber halo pulse, dual CTAs
2. **Signal vs Noise** — left flicker "noise", right stable "signal"
3. **System Shift** — icons/cards fade-in block styling
4. **Ecosystem Grid** — 6 brand cards, 3D tilt + hover glow
5. **DreamChamber** — fog + floating particles + 5-step flow
6. **GABRIEL** — minimal decision card, alive not chatty
7. **Economics** — "75% to the artist forever" + 3 feature tiles
8. **Final Silence** — black → silence close

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Cormorant Garamond (headlines) + Inter (UI)
- Respects `prefers-reduced-motion`

## Setup
```bash
cd apps/cathedral-web
npm install
npm run dev
```

## Deploy
- Vercel: connect this directory
- Or: `npx vercel --prod`
- Custom domain: `noizy.ai` (after CF email routing is set up)
